**RawenChat** es una app de escritorio para creadores de contenido en Twitch y Kick. Chat multiplataforma en tiempo real, comandos interactivos, TTS, avatar 2D reactivo y overlays para OBS. Cualquier espectador puede interactuar desde el día uno, sin necesidad de puntos de canal.

## Features

- **Chat Multiplataforma** — Conexión directa en tiempo real con Twitch y Kick.
- **Comandos Interactivos** — Macros de teclado y sonidos activados por comandos de chat.
- **Control Anti-Spam** — Sistema de timeouts personalizable por usuario.
- **Avatar 2D Reactivo** — Avatar VTuber que reacciona a la detección de sonido del micrófono.
- **Text-to-Speech** — El bot lee el chat en voz alta con control de idioma, voz y volumen.
- **Integración con OBS** — Genera links de overlays listos para pegar como Browser Source.
- **Chat Overlay Custom** — Overlay totalmente personalizable para que encaje con la identidad de tu stream.

## Tech Stack

- [Next.js 16](https://nextjs.org) (App Router + Turbopack)
- [React 19](https://react.dev)
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS v4](https://tailwindcss.com)

## Desarrollo

```bash
git clone https://github.com/rawencat/rawenchat-website.git
cd rawenchat-website
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.
